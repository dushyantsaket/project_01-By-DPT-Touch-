DPT Tools Touch
E-Commerce Website for Power Tools & Hardware
📌 Overview

DPT Tools Touch is a modern e-commerce web application designed for buying and selling power tools, hardware equipment, and industrial products.
It provides a smooth and user-friendly experience for customers to browse, search, and purchase tools online.

🚀 Features
🛒 Product listing with categories
🔍 Search and filter functionality
📦 Product details page
🧾 Add to cart & checkout system
👤 User authentication (Login / Signup)
💳 Secure payment integration (optional)
📱 Fully responsive design
🧑‍💼 Admin dashboard (optional)
Add / update / delete products
Manage orders
